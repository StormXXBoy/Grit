# Lua documentation

Welcome to the Lua documentation for **Grit**, the 2D platformer sandbox engine/game. This documentation covers the Lua API you can use to interact with the game, create gameplay mechanics, spawn entities, handle input, and implement custom behaviors.

Use this as a guide to:

* Get started with Lua scripting in Grit
* Learn about core lifecycle functions (`init`, `update`, `onInput`, `serverConnected`)
* Understand the Entity system and how to manipulate objects
* Handle networking between client and server
* Discover best practices for safe and stable scripting
